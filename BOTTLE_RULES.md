# Bottle (МДЧ) Rules

- Two players only:
  - Player A = who starts bottle
  - Player B = first who presses "Играть"
- Task must be completed in chat (text/voice/circle/photo)
- Result:
  - done -> +10 karma
  - fail -> -10 karma
- Cooldown: 5 minutes per chat
- Task pool is local (no AI), tuned for edgy friends-chat vibe.
