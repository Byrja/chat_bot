# 🤖 WebApp Мамули

Веб-приложение для Telegram бота с функциями:
- 📝 Анонимные признания
- 📊 Статистика пользователей 
- 👥 Оценка друзей и козлов
- 📚 Цитатник

## 🚀 Быстрый запуск

### Локальное тестирование

```bash
cd webapp
python start_local.py
```

Этот скрипт автоматически:
- ✅ Проверит базу данных
- ✅ Покажет инструкции по настройке бота
- ✅ Запустит веб-сервер на http://localhost:5001

### Ручной запуск

```bash
cd webapp
python app.py
```

## ⚙️ Настройка Telegram бота

### Вариант 1: Кнопка Menu

1. Откройте @BotFather в Telegram
2. Выберите бота: `/mybots`
3. `Bot Settings` → `Menu Button` → `Configure Menu Button`
4. URL: `http://localhost:5001` (для локального тестирования)
5. Название: `WebApp Мамули`

### Вариант 2: Команда в коде бота

```python
@bot.message_handler(commands=['webapp'])
def send_webapp(message):
    markup = types.InlineKeyboardMarkup()
    webapp_button = types.InlineKeyboardButton(
        text="Открыть WebApp",
        web_app=types.WebAppInfo(url="http://localhost:5001")
    )
    markup.add(webapp_button)
    bot.send_message(message.chat.id, "Нажмите кнопку:", reply_markup=markup)
```

## 🔧 Диагностика проблем

### Проблема: "Не удалось получить user_id из Telegram WebApp API"

**Возможные причины:**
- ❌ Приложение открыто в браузере, а не через Telegram
- ❌ WebApp не настроен в боте
- ❌ Неправильный URL в настройках

**Решения:**
1. Откройте WebApp только через Telegram (НЕ в браузере!)
2. Проверьте URL в настройках бота
3. Убедитесь, что бот запущен

## 📚 Цитатник

Веб-приложение теперь поддерживает функцию "Цитатник", которая позволяет просматривать "золотые фразы" участников чата.

### Функции:
- Просмотр случайных цитат из чата
- Ссылка на оригинальное сообщение в Telegram
- Обновление списка цитат

### Использование:
1. Откройте WebApp через Telegram
2. Нажмите кнопку "Цитатник" на главной странице
3. Просматривайте цитаты и переходите к оригинальным сообщениям

## ⚠️ Важные замечания

- **WebApp работает ТОЛЬКО в Telegram приложениях**
- Тестирование в обычном браузере невозможно
- Для продакшена требуется HTTPS (кроме localhost)
- База данных должна быть в родительской папке: `../chat.db`

## 🆘 Помощь

Если ничего не работает:

1. Проверьте, что файл `../chat.db` существует
2. Убедитесь, что бот запущен и отвечает
3. Попробуйте перезапустить веб-приложение
4. Проверьте настройки WebApp в @BotFather
5. Откройте консоль разработчика (F12) для проверки ошибок
